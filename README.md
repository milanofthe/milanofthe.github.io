# milanrother.com

Personal consulting website for Milan Rother - freelance consultant specializing in numerical simulation and scientific computing.

## Tech Stack

- **Framework**: SvelteKit with adapter-static
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site automatically deploys to GitHub Pages when pushing to the `main` branch via GitHub Actions.

## Features

- Animated Lorenz attractor visualization
- Phase Space color theme (teal/amber on charcoal)
- PathSim project showcase
- Services section
- Responsive design
- Static site generation for fast loading
